# Personal Expense Analytics

This project analyzes personal spending habits and visualizes expenses across different categories using Python.

## Tools Used

- Python
- Pandas
- Matplotlib

## Features

- Load expense data from a CSV file
- Calculate total expenses
- Analyze spending by category
- Visualize expenses using a bar chart

## How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the script:

python expense_analysis.py

## Output

The project generates a chart named `expense_analysis.png` showing expenses by category.
